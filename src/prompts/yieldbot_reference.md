## Project Overview
**Name:** Yieldbot  
**Website:** https://yieldbot.cc  
**Token:** $YBOT  

## What is Yieldbot?
Yieldbot is a DeFi protocol token controlled by smart contracts and AI. It helps users leverage AI agents on blockchain for:

- **Data & Questions**: AI-powered insights for blockchain queries
- **Yield Trading**: Automated AI agents execute yield trades to achieve high APY
- **Investment Returns**: All profits go back to the project, investors, and ecosystem growth
- **Credit Lending Pool**: Building a decentralized credit lending pool system
